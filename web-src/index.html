Public, read-only product and homepage tools for MCP Apps-compatible LLM hosts, with REST fallbacks for other integrations.

REST

GET /v1/products

GET /v1/products/{id}

GET /v1/whats-new

MCP

POST /mcp

Tools expose listing, detail, homepage summary, and rich render surfaces.

Config

Set SITE_INDEX_URL and SITE_BASE_URL before running or deploying.

Local Smoke Request

curl -s "$APP_URL/v1/products?category=credit%20cards" | jq

MCP Initialize

curl -s "$APP_URL/mcp" \
  -H "content-type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-03-26"}}' | jq
